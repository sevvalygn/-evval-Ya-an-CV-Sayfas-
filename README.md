  
# Şevval Yağan CV Sayfası

Bu proje, HTML ve CSS kullanılarak hazırlanmış kişisel CV sayfasıdır.  
Sayfa, profil bilgileri, eğitim geçmişi, iş deneyimi, yetenekler ve iletişim bilgilerini içermektedir.  

## Özellikler

- Profil fotoğrafı ve isim başlığı
- Özgeçmiş ve eğitim bölümleri
- İş deneyimi ve yetenekler listesi
- İletişim bilgileri ve GitHub/LinkedIn bağlantıları
- Stil ve renkler CSS ile düzenlenmiştir
- `<section>` ve `<div>` etiketleri kullanılarak bölümler oluşturulmuştur

## Kullanım

1. Bu depoyu bilgisayarınıza klonlayın veya indirin:
   ```bash
   git clone https://github.com/sevvalygn/cv-sayfa.git
   Dosya Yapısı
   cv-sayfa/
│
├─ index.html       # CV sayfası
├─ güncel.png       # Profil fotoğrafı
└─ README.md        # Proje açıklaması
